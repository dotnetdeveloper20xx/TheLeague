Project Overview

❓

The Problem

Sports clubs traditionally rely on spreadsheets, paper records, WhatsApp groups, and disconnected systems to manage memberships, bookings, events, and finances. This leads to data inconsistency, administrative overhead, poor member experience, and difficulty tracking payments.

✓

The Solution

The League provides a unified platform with three distinct portals - Super Admin for platform management, Club Manager for operational control, and Member Portal for self-service. It automates member lifecycle management, session booking with capacity control, event ticketing, and payment tracking.

👥

Target Users

  • Platform Admins - Manage multiple clubs and system configuration
  • Club Managers - Handle daily operations, members, sessions, events
  • Members - Self-service for bookings, payments, and family management
